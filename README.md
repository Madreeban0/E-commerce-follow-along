# E-commerce-follow-along
# Ecommerce follow project

Welcome to the ** Ecommerce follow along project **, this is a hands-on project where will build a e-commercd application using the MERN stack (MongoDB, Express.js, React.js and node.js). The goal is to learn how to develop a full stack web application step by step.

# Milestone 1: Project Overview

#### 1.Unerstanding the MERN stack:
**MongoDB:** A database for storing application data in felxible, document based format.
**Express.js:** A backend web application framework for building APIs and handling server logics.
**REact.js:** A framework library for building User Interfaces.
**Node.js:** aruntime environment theat allows javascript to run on the server.

#### 2.  REST API Structure
REST APIs are used to allow communication between the front-end and backend.
we'll create APIs for:
_ **User Authentication**: Allowing users to register and log in.
_ **Product Management**: Adding. updating , and retirving product data.
_ **Order Handling**: Managing customer orders.

#### 3.  Databases Schema Design
We'll learn how to design and organize using MongoDB. A schema helps us define hoe the data is stored and related. 

#### 4.Authentication 
Authentication ensures only the right peaple can access certain features. For example:
- Users need log in to place orders or see their personal data.
- It keeps the app secure by verfying users identities.
